    Speed up map loading. The change with the most impact, especially on systems
    with slow graphics drivers, is the removal of continuous updates of the
    progress bar while loading 2D and 3D objects. Further optimizations include:
    * Keep the object and texture caches sorted, and use a binary search instead
      of looping over the entire cache to find an object.
    * Same for the list of harvestable and entrable objects.
    * Keep a counter on where to insert the next 2D/3D object corresponding
      object list, instead of (again) looping over the list to find the next empty
      slot.
    * Get rid of the use of get_string_occurance() and friends when parsing 2d0
      files. Instead of searching through the whole file for every field name,
      simply walk through the file and process the keys we recognize.
